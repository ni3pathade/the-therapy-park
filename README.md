Therapy Website Redesign – Custom WordPress Theme

This project is a custom WordPress website developed based on the provided Figma design. 
The theme is built using PHP templates, HTML5, CSS3, and custom JavaScript.

The website includes:
- Custom theme development (no page builders)
- Dynamic Blog system
- Team Members & Testimonials (CPT + ACF)
- AJAX-based blog filtering
- Calendly booking integration
- Mailchimp newsletter integration
- Fully responsive layout

Technologies Used:
- PHP (WordPress Template Hierarchy)
- HTML5
- CSS3
- JavaScript
- ACF (Advanced Custom Fields)
- CPT UI
- WPForms
